Anatomia de um formulário Fluig

Um formulário Fluig É HTML — com convenções obrigatórias. Sem elas, os dados não salvam e os estilos não funcionam.

<div class="fluig-style-guide">Wrapper obrigatório
<div class="panel panel-default">Agrupamento visual
<div class="row">Linha do grid
<div class="col-md-6">Coluna (soma = 12)
<div class="form-group">Container do campo
<label> + <input name="...">Label + campo com name
<div class="fluig-style-guide" style="padding: 20px;">
  <div class="panel panel-default">
    <div class="panel-heading">
      <h3 class="panel-title">Título</h3>
    </div>
    <div class="panel-body">
      <div class="row">
        <div class="col-md-6">
          <div class="form-group">
            <label for="campo">Label</label>
            <input type="text" class="form-control"
                   id="campo" name="campo">
          </div>
        </div>
      </div>
    </div>
  </div>
</div>
Regras do atributo name
RegraCorretoErrado
Snake casenome_completonomeCompleto
Sem acentosdescricaodescrição
Sem espaçosdata_nascimentodata nascimento
Sem hífenscentro_custocentro-custo

Campo sem name = dado perdido. Campo disabled = dado NÃO enviado (use readonly para manter o dado).
Elementos
Todos os tipos de input
TipoHTMLSalva como
Texto<input type="text" name="x">String
Email<input type="email" name="x">String (validado)
Número<input type="number" name="x">String
Data<input type="date" name="x">YYYY-MM-DD
Select<select name="x"><option value="v">O value da option
Radio<input type="radio" name="x" value="v">Value do selecionado
Checkbox<input type="checkbox" name="x" value="S">Só se marcado
Textarea<textarea name="x">String multilinha
Hidden<input type="hidden" name="x" value="v">String (invisível)
Select: O value da <option> é o que salva no banco, não o texto visível. Radio: Todos com o mesmo name formam grupo — só um pode ser selecionado. Checkbox: Só envia o valor se estiver marcado.

Validação jQuery
$('#btn-salvar').click(function() {
    var erros = [];

    if ($('#nome').val().trim() === '') {
        erros.push('Nome é obrigatório');
    }
    if ($('#departamento').val() === '') {
        erros.push('Selecione um departamento');
    }
    if (!$('input[name="aceite"]').is(':checked')) {
        erros.push('Aceite os termos');
    }

    if (erros.length > 0) {
        alert('Corrija:\\n- ' + erros.join('\\n- '));
        return false;
    }
});

    // Preencher nome do solicitante

    // Preencher data com hoje
    var hoje = new Date().toISOString().split('T')[0];
    $('#data_solicitacao').val(hoje);

Troubleshooting
ProblemaSolução
Campo não salva dadosVerificar atributo name
Select salva texto em vez de códigoVerificar value de cada <option>
Radio não funciona como grupoTodos do grupo com mesmo name
Checkbox não salva quando desmarcadoComportamento padrão — tratar no servidor
Campo disabled não salvaUsar readonly em vez de disabled
Estilo não aplicaVerificar wrapper fluig-style-guide
Máscara não funcionaIncluir jquery.mask.min.js no resources/
