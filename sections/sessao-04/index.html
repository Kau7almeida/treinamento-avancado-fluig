Formulários Avançados e Técnicas Dinâmicas

Zoom, AutoComplete, Pai × Filho, Select com Dataset e Listas Dinâmicas — os recursos que transformam formulários simples em soluções corporativas.

Busca avançada
Campo Zoom

Zoom é um campo de busca avançada com modal pesquisável. O usuário digita, a modal abre com resultados de um dataset, seleciona, e os campos do formulário são preenchidos automaticamente.

// Configuração do Zoom
var opts = {
    source: 'ds_fornecedores',          // Dataset fonte
    displayKey: 'cod_fornecedor',       // Campo exibido no input
    formFieldsList: [                   // Mapeamento dataset → form
        { textFieldDatasetName: 'codigo', textFieldFormName: 'cod_fornecedor' },
        { textFieldDatasetName: 'nome',   textFieldFormName: 'nome_fornecedor' },
        { textFieldDatasetName: 'cnpj',   textFieldFormName: 'cnpj_fornecedor' }
    ],
    tableHeader: ['Código', 'Nome', 'CNPJ'],
    tableFieldsList: ['codigo', 'nome', 'cnpj'],
    maxResult: 10
};

FLUIGC.zoom('#cod_fornecedor', opts);

// Eventos
$('#cod_fornecedor').on('fluig.zoom.itemSelect', function(e, item) {
    console.log('Selecionado:', item);
});
$('#cod_fornecedor').on('fluig.zoom.itemRemove', function(e) {
    $('#nome_fornecedor').val('');  // limpar campos
});
AutoComplete

Irmão mais simples do Zoom — mostra sugestões inline conforme digita, sem modal.

var opts = {
    source: 'ds_produtos',
    displayKey: 'descricao',
    minLength: 2,   // mínimo de chars para buscar
    maxResult: 10
};
FLUIGC.autocomplete('#produto', opts);
Cascateamento
Técnica Pai × Filho

Selecionar um campo (pai) filtra automaticamente as opções de outro (filho). Exemplo: Estado → Cidade, Departamento → Setor, Categoria → Subcategoria.

$('#estado').change(function() {
    var uf = $(this).val();
    var $cidade = $('#cidade');

    $cidade.empty();  // OBRIGATÓRIO: limpar opções anteriores!

    if (!uf) {
        $cidade.append('<option value="">Selecione o estado primeiro...</option>');
        $cidade.prop('disabled', true);
        return;
    }

    // Buscar cidades via dataset com filtro
    var constraints = [];
    constraints.push(
        DatasetFactory.createConstraint("uf", uf, uf, ConstraintType.MUST)
    );
    var dados = DatasetFactory.getDataset("ds_cidades", null, constraints, null);

    $cidade.append('<option value="">Selecione...</option>');
    for (var i = 0; i < dados.values.length; i++) {
        $cidade.append(
            '<option value="' + dados.getValue(i, "codigo") + '">' +
            dados.getValue(i, "nome") + '</option>'
        );
    }
    $cidade.prop('disabled', false);  // Habilitar
});
Bug clássico: Não limpar o campo filho quando o pai muda. Resultado: o usuário seleciona "Rio de Janeiro" mas a cidade ainda mostra "Campinas" (de São Paulo). Sempre use $filho.empty() antes de popular.

Tabela filha
Listas Dinâmicas

Tabelas onde o usuário adiciona e remove linhas. Usa a convenção dos 3 underscores (___) para criar tabela filha automática no banco.

Convenção ___ (3 underscores): name="item_descricao___1" = linha 1, coluna descricao. name="item_descricao___2" = linha 2. O Fluig cria a tabela filha automaticamente no banco de dados.
HTML da tabela
<table class="table table-striped" id="tabela-itens">
  <thead>
    <tr>
      <th>#</th><th>Descrição</th><th>Qtd</th>
      <th>Valor</th><th>Subtotal</th><th></th>
    </tr>
  </thead>
  <tbody id="corpo-tabela">
    <tr class="linha-item" data-linha="1">
      <td class="num-linha">1</td>
      <td><input class="form-control input-sm"
               name="item_descricao___1"></td>
      <td><input class="form-control input-sm campo-qtd"
               name="item_quantidade___1" value="1"></td>
      <td><input class="form-control input-sm campo-valor"
               name="item_valor___1"></td>
      <td><input class="form-control input-sm campo-subtotal"
               name="item_subtotal___1" readonly></td>
      <td><button class="btn btn-danger btn-xs btn-remover">✕</button></td>
    </tr>
  </tbody>
</table>

Remover + Reindexar
// Remover
$(document).on('click', '.btn-remover', function() {
    if ($('.linha-item').length <= 1) {
        alert('Mínimo 1 item');
        return;
    }
    $(this).closest('tr').remove();
    reindexarLinhas();  // OBRIGATÓRIO!
});

// Reindexar (corrigir ___N após remoção)
function reindexarLinhas() {
    $('.linha-item').each(function(index) {
        var num = index + 1;
        $(this).find('.num-linha').text(num);
        $(this).find('input[name^="item_descricao___"]')
               .attr('name', 'item_descricao___' + num);
        $(this).find('input[name^="item_quantidade___"]')
               .attr('name', 'item_quantidade___' + num);
        $(this).find('input[name^="item_valor___"]')
               .attr('name', 'item_valor___' + num);
        $(this).find('input[name^="item_subtotal___"]')
               .attr('name', 'item_subtotal___' + num);
    });
    contador = $('.linha-item').length;
}
Cálculo automático
// Subtotal por linha (qtd × valor)
$(document).on('change keyup', '.campo-qtd, .campo-valor', function() {
    var $linha = $(this).closest('tr');
    var qtd = parseFloat($linha.find('.campo-qtd').val()) || 0;
    var valorStr = $linha.find('.campo-valor').val() || '0';
    var valor = parseFloat(valorStr.replace(/\./g,'').replace(',','.')) || 0;
    var subtotal = qtd * valor;
    $linha.find('.campo-subtotal').val(
        subtotal.toLocaleString('pt-BR', {minimumFractionDigits:2})
    );
    calcularTotal();
});

// Total geral
function calcularTotal() {
    var total = 0;
    $('.campo-subtotal').each(function() {
        var str = $(this).val() || '0';
        total += parseFloat(str.replace(/\./g,'').replace(',','.')) || 0;
    });
    $('#total_geral').val(
        total.toLocaleString('pt-BR', {minimumFractionDigits:2})
    );
}

Quando usar o quê?
Cenário Componente
2-4 opções fixas (Sim/Não, prioridade) Radio
5-50 opções estáticas Select fixo no HTML
5-50 opções que podem mudar Select com Dataset
50+ opções (fornecedores, produtos) Zoom
Busca rápida com sugestão AutoComplete
Campos dependentes (Estado→Cidade) Pai × Filho
Múltiplos registros (itens do pedido) Lista Dinâmica

Troubleshooting
Problema Solução
Zoom abre modal vazia Verificar nome do dataset (case sensitive)
Zoom não preenche campos Conferir formFieldsList (dataset → form)
Pai×Filho: valor antigo persiste Adicionar $filho.empty() antes de popular
Select mostra "Carregando..." Dataset não carregou — ver console F12
Lista perde dados ao salvar Verificar padrão nome___N
Lista após remover: dados trocados Chamar reindexarLinhas() após remover
Cálculo retorna NaN Converter moeda: replace('.','').replace(',','.')
