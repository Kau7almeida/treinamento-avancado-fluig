Estrutura de projetos
meu-projeto/
├── forms/              ← Formulários HTML
│   └── frm_hello_fluig/
│       └── frm_hello_fluig.html
├── workflow/           ← Processos BPMN (.process)
├── datasets/           ← Datasets customizados (.js)
├── wcm/                ← Portais e Widgets
│   ├── widget/
│   └── layout/
├── events/             ← Eventos globais
└── resources/          ← Imagens, CSS, JS compartilhados
Regra de ouro: A pasta define o tipo do artefato. Formulário fora de forms/ não faz deploy como formulário. O Studio identifica o tipo pela localização do arquivo.
Design system
fluig Style Guide

O Style Guide é o catálogo de componentes visuais do Fluig. Garante consistência, simplifica manutenção, e mantém compatibilidade com atualizações da plataforma. Use sempre — nunca reinvente a roda.

Grid System — 12 colunas

Toda organização de layout usa o grid de 12 colunas. col-md-* para desktop, col-sm-* para tablet, col-xs-* para mobile. As colunas somam 12 por linha.

<!-- 2 colunas iguais -->
<div class="row">
    <div class="col-md-6">Coluna 1</div>
    <div class="col-md-6">Coluna 2</div>
</div>

<!-- 3 colunas -->
<div class="row">
    <div class="col-md-4">1/3</div>
    <div class="col-md-4">1/3</div>
    <div class="col-md-4">1/3</div>
</div>

<!-- Responsivo: 2 col desktop, 1 col mobile -->
<div class="row">
    <div class="col-md-6 col-xs-12">Adaptável</div>
    <div class="col-md-6 col-xs-12">Adaptável</div>
</div>

Troubleshooting
Problema Causa provável Solução
java -version não encontrado Java não instalado Instalar JDK 8 ou 11, configurar JAVA_HOME
Plugin não aparece após install Eclipse não reiniciou ou URL errada Reiniciar Eclipse; verificar URL do update site
Connection refused URL/porta errada ou firewall Verificar http vs https, porta, regras de firewall
Deploy sem erro mas form não aparece Cache do browser Ctrl + Shift + Del ou Ctrl + F5
Deploy falha com erro de permissão Usuário sem role de dev Solicitar perfil admin/desenvolvedor
Eclipse muito lento Memória insuficiente eclipse.ini: -Xmx1024m → -Xmx2048m
Style Guide não carrega Site fora ou bloqueado Verificar acesso a style.fluig.com
